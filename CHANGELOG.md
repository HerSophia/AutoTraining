# CHANGELOG — AutoTraining

版本号遵循语义化版本。适配游戏：Esports Manager 2026 Steam 正版补丁 1.0.1（Unity 6000.3.12f1 / IL2CPP metadata v39）。

## 1.0.0（2026-07-12）

- 首个对外发布版本。
- 为每个玩家队自动安排地图训练：每张地图可设核心 / 维护 / 禁练 / 未配置四档，插件在每天地图训练结算后用游戏自身接口挑选后续训练地图。
- 全队无任何正向配置时保持休眠，不改动原版训练计划。
- 训练槽上限 3 张图；默认核心目标 85%、维护目标 70%、最短持有期 7 个游戏日。
- 内嵌进原生地图训练页的「自动训练」区块；内嵌注入失败时降级为可拖动的悬浮面板（默认 F8）。
- 插件 UI 支持中英文，随游戏语言切换实时刷新。
- 依赖 EM.Framework 共享框架。
